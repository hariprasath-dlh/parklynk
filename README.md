# ParkLynk

**"Linking Empty Spaces to Moving Lives."**

ParkLynk is a peer-to-peer parking marketplace platform built using the MERN stack. It connects parking space owners with drivers looking for convenient, affordable parking.

---

## 📁 Project Structure

```
parkklynk/
│
├── instruction/          # AI agent development instructions
├── skills.md             # Business features & functional requirements
├── technical-stack.md    # Locked technical stack decisions
├── backend/              # Node.js + Express API server
├── frontend/             # React (Vite) client application
└── README.md             # Project overview
```

---

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+)
- MongoDB Atlas account
- Razorpay test credentials
- Cloudinary account

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
